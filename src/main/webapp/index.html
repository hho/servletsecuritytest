Test Basic Auth

This dummy app has a servlet mapped on /api/*.

You can access the API freely, except for the endpoints under /api/devicemanagement/gateways, those need authentication.

Definition in web.xml: 

	<security-constraint>
		<web-resource-collection>
			<web-resource-name>Special API methods</web-resource-name>
			<url-pattern>/api/devicemanagement/gateways/*</url-pattern>
		</web-resource-collection>
		<auth-constraint>
			<role-name>user</role-name>
		</auth-constraint>
	</security-constraint>

Examples

/api/some-endpoint
should be available without authentication
/api/devicemanagement/gateways/add
should be protected
/api//devicemanagement/gateways/add
should also be protected – but in Jetty, it's not!
